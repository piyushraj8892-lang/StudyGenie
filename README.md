# 🎓 StudyGenie AI

StudyGenie AI is an educational platform built with Python and Streamlit that helps students create personalized homework, assignments, projects, essays, notes, and other academic materials using artificial intelligence.

The goal of this project is to make learning easier by providing content that matches a student's class level, subject, and requirements.

## ✨ Features

* Homework Generator
* Assignment Generator
* Project Creator
* Essay Writer
* Notes Generator
* Class-based personalization
* Support for school and college students

## 🛠 Built With

* Python
* Streamlit
* Gemini AI
* SQLite (planned)
* Razorpay (planned)

## 🚀 Future Features

* User Login & Signup
* 15-Day Free Trial
* Monthly Subscription Plans
* PDF & DOCX Downloads
* Portfolio Builder
* PPT Content Generator

## 📚 Purpose

This project aims to help students learn and organize their academic work more efficiently using AI.
